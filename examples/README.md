# Examples

These scripts demonstrate the intended workflow, but the experimental data files are not included in this public repository.

Before running an example, place your private data files in your MATLAB working directory or update the filenames inside the script.

Expected private data examples:

- `CO2_850C.txt`
- `CO2_900C.txt`
- `CO2_950C.txt`
- `H2O_850C.txt`
- `Mix_850C_5gH2O_100mLCO2.txt`
- `pulse_data_11.txt`

Run from the repository root:

```matlab
startup
RunSingleTGAExample
```
