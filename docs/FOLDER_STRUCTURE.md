# Folder Structure

```text
src/
```

Core scientific workflow functions: conversion calculation, isothermal segment extraction, and synergy-factor analysis.

```text
fitting/
```

RPM fitting and temperature-dependent kinetic analysis.

```text
simulation/
```

Pulsed gasification prediction and measured-vs-predicted validation logic.

```text
utilities/
```

File readers, plotting style helpers, figure export, and table export helpers.

```text
examples/
```

Runnable workflow scripts. They require private data files that are not included.

```text
figures/
```

Plotting functions only. Experimental images are not included.

```text
archive/
```

Historical, data-specific, and thesis-figure scripts retained for traceability but not part of the main public API.
