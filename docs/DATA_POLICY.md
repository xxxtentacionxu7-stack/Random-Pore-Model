# Data Policy

Experimental data are intentionally omitted from this public repository.

Do not commit:

- Raw TGA text files.
- Excel workbooks exported from instruments or Origin.
- CSV source-data files derived from experiments.
- MAT files containing fitted or intermediate experimental results.
- Unreviewed publication figures.

The example scripts still show the expected filenames and workflow structure. To run them, place private data files locally and keep them outside Git tracking.
